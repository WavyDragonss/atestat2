Tipuri de reactii chimice

O reactie chimica este un proces in care una sau mai multe substante (reactanti) se transforma in alte substante (produsi) cu proprietati diferite. Reactiile sunt clasificate dupa modul in care atomii se rearanjeaza. Iata cele cinci tipuri principale.

Atentie: reactiile chimice pot elibera caldura, lumina sau gaze. Experimentele se efectueaza doar sub indrumarea unui profesor.

Reactii dupa modul de rearanjare a atomilor

Reactia de combinare (sinteza)

Doua sau mai multe substante se unesc si formeaza un singur produs. Forma generala: A + B → AB. Exemplu: arderea hidrogenului in oxigen pentru a forma apa — 2H2 + O2 → 2H2O.

Reactia de descompunere

O singura substanta se rupe in doua sau mai multe substante mai simple, de obicei sub actiunea caldurii sau curentului electric: AB → A + B. Exemplu: descompunerea apei oxigenate — 2H2O2 → 2H2O + O2.

Reactia de substitutie

Un element mai reactiv il „inlocuieste" pe altul, mai putin reactiv, dintr-un compus: A + BC → AC + B. Exemplu: zincul scoate cuprul din sulfat de cupru — Zn + CuSO4 → ZnSO4 + Cu.

Reactia de schimb (dublu schimb)

Doi compusi isi schimba intre ei „partile" — anionii si cationii trec dintr-un compus in celalalt: AB + CD → AD + CB. Exemplu: formarea unui precipitat — AgNO3 + NaCl → AgCl↓ + NaNO3.

Reactii dupa schimbul de electroni si proton

Reactii de oxido-reducere (redox)

Au loc cu transfer de electroni: o substanta se oxideaza (cedeaza electroni), alta se reduce (accepta electroni). Exemplu: ruginirea fierului — 4Fe + 3O2 → 2Fe2O3.

Reactii acid-baza (neutralizare)

Un acid si o baza reactioneaza si formeaza o sare si apa: HCl + NaOH → NaCl + H2O. Practic, ionii H+ si OH− se unesc.

Reactii de ardere

O substanta reactioneaza rapid cu oxigenul, eliberand caldura si lumina. Pentru combustibili organici: CH4 + 2O2 → CO2 + 2H2O (arderea metanului — flacara aragazului).

Reactii exoterme & endoterme

Exoterme elibereaza energie sub forma de caldura (ex. arderile). Endoterme absorb energie din mediu (ex. fotosinteza: 6CO2 + 6H2O → C6H12O6 + 6O2).

Concluzie

Cele cinci tipuri principale de reactii chimice (combinare, descompunere, substitutie, schimb, redox) descriu majoritatea transformarilor materiei. Recunoasterea tipului unei reactii ne ajuta sa prezicem produsii si sa scriem corect ecuatia chimica, respectand legea conservarii masei.
